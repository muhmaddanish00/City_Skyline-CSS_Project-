# 🌆 City Skyline Project

A beautiful and responsive City Skyline built using pure HTML and CSS.  
This project demonstrates advanced CSS concepts like gradients, CSS variables, and media queries.

---

## 🚀 Features

- 🌇 Realistic sky using radial-gradient
- 🏢 Buildings created with CSS
- 🎨 CSS Variables for theme colors
- 📱 Fully Responsive (Media Queries)
- 🌙 Dark mode effect on smaller screens

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Gradients
- CSS Variables
- Media Queries
